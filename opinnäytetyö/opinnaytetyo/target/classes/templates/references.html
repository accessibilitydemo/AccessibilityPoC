This web-site has been created as a part of a thesis focusing on accessibility on online forms. The aim of this page is to provide simple proof of concept / demo on some accessibilities with online form. For this reason I have created a simple "Research grant for school personnel" form that starts from the next page.

The form in question has simple guidance on what needs to be filled as well as error messaging and highlighting empty fields with red colors. The point of the form page is not to let the user "submit" it before all the required fields are filled. To move on to the form page, please press button next (blue) at the end of this page. To logout press logout button (red). For further information on accessibility and how this website is created below.


In terms of accessibility the form show the user where they are within the website / process (navigator), it also indicates what fields need to be filled and which are obligatory, if the user fails to fill an obligatory field it will not let the user move on before filling it, the form also offers simple guidance and before it can be send you will have the possibility to go through it once more in form of a summary.

The form itself is not aiming to replicate real life situation in terms of asking everything required from the user to apply for the grant. It is only meant to showcase some accessibility features to support the thesis.
